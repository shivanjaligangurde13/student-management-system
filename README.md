# Student Management System

A RESTful backend application built using Java, Spring Boot, MySQL, and Spring Data JPA.

## Features

- Add Student
- View Students
- Update Student
- Delete Student

## Technologies Used

- Java 17
- Spring Boot
- MySQL
- Spring Data JPA
- Maven
- Postman

## API Endpoints

GET /students

GET /students/{id}

POST /students

PUT /students/{id}

DELETE /students/{id}
